Presets are the heart of UnoCSS. They bundle rules, variants, and theme configuration into reusable packages. You can combine multiple presets in a single project, and the engine merges them together on demand.

## Adding presets to your config

Install the preset package, then register it in your `uno.config.ts`:

```ts
import presetWind3 from '@unocss/preset-wind3'
import presetIcons from '@unocss/preset-icons'
import { defineConfig } from 'unocss'

export default defineConfig({
  presets: [
    presetWind3(),
    presetIcons(),
    // ...add more presets here
  ],
})
```

<Tip>
All official presets are also re-exported from the `unocss` package, so you can import from a single location:

```ts
import { presetWind3, presetIcons, presetTypography } from 'unocss'
```
</Tip>

### All official packages

| Package | Description |
|---|---|
| `@unocss/preset-mini` | The minimal but essential rules and variants |
| `@unocss/preset-wind3` | Tailwind CSS / Windi CSS compact preset |
| `@unocss/preset-wind4` | Tailwind CSS v4 compact preset (experimental) |
| `@unocss/preset-attributify` | Enables attributify mode for other rules |
| `@unocss/preset-tagify` | Enables tagify mode for other rules |
| `@unocss/preset-icons` | Pure CSS icons powered by Iconify |
| `@unocss/preset-web-fonts` | Web fonts (Google Fonts, etc.) support |
| `@unocss/preset-typography` | Typography prose classes |
| `@unocss/preset-rem-to-px` | Converts rem to px for utilities |

<Note>
`@unocss/preset-wind` and `@unocss/preset-uno` are deprecated and have been renamed to `@unocss/preset-wind3`.
</Note>

## Tagify mode

`@unocss/preset-tagify` enables tagify mode — where you use a utility class name directly as an HTML tag. This is useful for single-utility elements.

```ts
import { defineConfig, presetTagify } from 'unocss'

export default defineConfig({
  presets: [
    presetTagify({ /* options */ }),
  ],
})
```

With tagify mode enabled, you can write:

```html
<text-red> Red text </text-red>
<flex> Flex container </flex>
<p2> 8px padding element </p2>
```

Instead of:

```html
<span class="text-red"> Red text </span>
<div class="flex"> Flex container </div>
<p class="p-2"> 8px padding element </p>
```

Use the `prefix` option to namespace your tag utilities:

```ts
presetTagify({
  prefix: 'un-',
  // <un-flex>, <un-text-red>, etc.
})
```
