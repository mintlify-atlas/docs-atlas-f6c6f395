## Welcome to UnoCSS

UnoCSS is an instant, on-demand atomic CSS engine designed to be flexible and extensible. Unlike traditional CSS frameworks, UnoCSS has no core utilities — all CSS utilities are provided via composable presets, giving you complete control over your styling system.

Whether you're migrating from Tailwind CSS, building a custom design system, or starting fresh, UnoCSS adapts to your workflow.

## Get started in minutes

<Steps>
  <Step title="Install UnoCSS">
    Add UnoCSS to your project using your preferred package manager and build tool integration.

    ```bash
    npm install -D unocss
    ```
  </Step>
  <Step title="Configure your project">
    Create a `uno.config.ts` file and choose your presets.

    ```ts uno.config.ts
    import { defineConfig, presetWind3 } from 'unocss'

    export default defineConfig({
      presets: [
        presetWind3(),
      ],
    })
    ```
  </Step>
  <Step title="Add the plugin to your build tool">
    Wire up UnoCSS in your build tool (example for Vite):

    ```ts vite.config.ts
    import UnoCSS from 'unocss/vite'
    import { defineConfig } from 'vite'

    export default defineConfig({
      plugins: [
        UnoCSS(),
      ],
    })
    ```
  </Step>
  <Step title="Import the CSS entry">
    Add the UnoCSS virtual import to your main entry file.

    ```ts main.ts
    import 'virtual:uno.css'
    ```
  </Step>
</Steps>

## Why UnoCSS?

<CardGroup cols={3}>
  <Card title="Instant" icon="bolt">
    No parsing, no AST, no file scanning. UnoCSS is up to 5x faster than Windi CSS or Tailwind JIT.
  </Card>
  <Card title="Lightweight" icon="feather">
    Zero dependencies and browser-friendly at ~6kb min+brotli.
  </Card>
  <Card title="Fully customizable" icon="paintbrush">
    No hardcoded utilities. Define your own rules, shortcuts, and presets for any design system.
  </Card>
  <Card title="Pure CSS Icons" icon="icons">
    Use any icon from Iconify as a single CSS class — no JavaScript required.
  </Card>
  <Card title="Rich integrations" icon="plug">
    First-class support for Vite, Nuxt, Astro, Webpack, PostCSS, and CDN runtime.
  </Card>
  <Card title="Developer tools" icon="wrench">
    Interactive playground, VS Code extension, inspector, and JetBrains plugin included.
  </Card>
</CardGroup>
